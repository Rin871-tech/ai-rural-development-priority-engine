# AI-Based Rural Development Priority Engine 

## Overview
This project is an AI-powered decision support system designed to help government authorities prioritize rural development problems and evaluate the effectiveness of existing government schemes.

The platform identifies high-impact rural issues, explains why they persist, and recommends data-driven budget allocation at district and taluka levels.

## Key Features
- District Priority Index (DPI)
- Taluka-level problem prioritization
- Scheme Intelligence Module (coverage gap & performance status)
- Explainable AI decision logic
- Budget allocation recommendations

Supports outcome-oriented governance under the Viksit Bharat vision.



## Tech Stack
- Backend: FastAPI, Python, Pandas
- Frontend: React, Recharts
- AI Logic: Multi-factor weighted prioritization (Explainable)
